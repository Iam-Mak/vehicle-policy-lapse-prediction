# Roadmap

## Completed

* Built end-to-end training pipeline (ingestion, transformation, model training, evaluation)
* Implemented inference pipeline with reusable preprocessing and model artifacts
* Exposed prediction service using FastAPI
* Containerized services using Docker and orchestrated with docker-compose
* Deployed application on Azure Web App
* Set up basic CI pipeline for build and testing

## Next Steps

* Improve model performance using tree-based models (XGBoost / LightGBM)
* Add experiment tracking and model versioning (MLflow)
* Introduce data and model version control (DVC)
* Implement basic monitoring for prediction drift and performance
* Enhance feature engineering and feature selection

## Longer Term

* Introduce automated retraining pipeline
* Add explainability layer for model predictions
* Extend system to support batch and real-time scoring
* Improve system robustness and error handling for production usage
